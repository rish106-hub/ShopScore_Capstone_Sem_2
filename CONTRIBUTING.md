# 🛒 Contributing to ShopScore

Welcome! 👋  
Thanks for considering a contribution to **ShopScore**, our semester capstone project. Whether you're fixing bugs, writing docs, improving UI, or suggesting ideas — we value your input.

---

## 📌 What is ShopScore?

ShopScore is a user-focused product review and rating system built to help users make better shopping decisions.  
Your contributions will help make it faster, smarter, and easier to use.

---

## 🛠️ How to Contribute

Follow these simple steps to get started:

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR-USERNAME/ShopScore_Capstone_Sem_2.git
cd ShopScore_Capstone_Sem_2
```
###2. Create a branch
```bash
git checkout -b feature/your-feature-name
```
###3. Make Your Changes
	•	Code, documentation, bug fixes — all welcome.
	•	Stay consistent with the existing code style.
###4. Commit Changes
```bash
git commit -m "Add: meaningful message"
```
###5. Push & Open a PR
```bash
git push origin feature/your-feature-name
```
Go to GitHub and create a Pull Request (PR) targeting the main branch.
