---

## 🐞 Reporting a Vulnerability

If you discover a security vulnerability in ShopScore, please report it responsibly.

- **Email us at**: rishavdewan10@gmail.com  
- **Response Time**: Within 72 hours  
- **Updates**: We will provide a status update on your report within 5 working days.

Please include:
- A clear description of the issue
- Steps to reproduce it (if possible)
- Suggested fix or patch (if available)
